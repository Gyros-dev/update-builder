Дополнительные блоки

Скачанный HTML автономный: логотип и скриншоты встраиваются внутрь файла. Кнопка «Копировать HTML-код» помещает полный код письма в буфер обмена — его можно вставить в режим редактирования исходного HTML письма. Кнопка PDF откроет системное окно печати — выберите «Сохранить как PDF».

Как отправить письмо через VK Почту
Инструкция работает только в браузерной версии VK WorkSpace. Перед отправкой проверьте письмо в тестовой рассылке.
  1. Сформируйте письмо и нажмите кнопку «Копировать HTML-код».
  2. Создайте новое письмо в VK Почте. В пустой области письма нажмите правой кнопкой мыши и выберите «Исследовать элемент» / Inspect Element.
  3. В панели Elements найдите редактируемый блок письма — элемент <div><br></div>. Нажмите по нему правой кнопкой мыши и выберите «Edit as HTML».
  4. Вставьте скопированный HTML-код перед </div> и примените изменение. Закройте инструменты разработчика и проверьте письмо.
